mikrotik

1.Masukkan ke winbox,pilih mikrotik yang mau di setting di click lalu dan tekan connect di pojok kanan atas

2.masukkan ke wireles di samping kiri,masuk ke wifi interfaces,klik pada wlan1 lalu aktifkan dengan mengklik tanda centang di atas

3.pilih wireless,lalu pada mode nya di ganti ke ‘station’.band nya ‘2GHz-B/G/N.yang sesuai dengan foto,elanjutnya klik ‘scan’ di samping jika sudah Interface: ‘wlan1’ lalu klik start

4.Pilih wifi yang akan digunakan,lalu clik connect pada samping kanan atas

5.masuk lagi ke menu wireles disamping,masuk ke security profile klik tanda + di samping kiri atas,jika sudah diklik pada bagian general karena wifi yang saya gunakan tidak menggunakan password maka saya langsung saja klik ‘apply’ lalu ‘ok’

6.Langkah selanjutnya masuk ke IP lalu ‘address’ jika sudah dibagian addres list tambahkan IP disini saya memberikan IP 192.168.10.1/24 lalu di interface dipilih ‘ether2’ lali klik apply lalu ok pada pojok kanan

7.selanjutnya masukkan ke IP pada pojok kiri lalu pilih ‘DHCP client’ jika sudah klik tanda + jika sudah di interface pilih ‘wlan1’ lalu klik apply lalu ok

8.masukkan ke IP lagi dikanan lalu pilih ‘DHCP server’ lalu pilih DHCP setup untuk DHCP server interface nya pilih ‘ether2’ lalu klik next sampai selesai

9.masuk ke IP lagi lalu pilih yang ‘firewall’ lalu pilih NAT klik tanda +

10.setelah di klik tanda + lalu pilih action pada bagian action pilih yang ‘masquerade’ jika sudah lalu klik ok

11.selanjutnya pergi ke new terminal di pojok samping bawah,untuk mengetes apakah sudah mendapatkan jaringannya apa belum.setelah itu ketik ping google.com jika berhasil maka tampilan nya seperti pada foto dibawah

12.Untuk mengetes apakah jaringan sudah bisa digunakan apa tidak,kita pergi ke browser lalu ketik yt jika terbuka dan berhasil maka tampilan nya seperti dibawah.ok untuk konfigurasi mikrotik dalam mode station sudah selesai.
