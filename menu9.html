Konfigurasi mikrotik di winbox mode ap bridge

Dalam mode ap bridge saya membuat mikrotik dalam mode station dulu,untuk menangkap jaringan, lalu saya sebarkan menggunakan mode ap bridge. Langkah pertama masuk ke tampilan awal winboxnya lalu klik tanda biru seperti dibawah lalu connect,pergi wireless lalu,klik wlan1 seperti di foto kanan lalu klik tanda centang

2.pergi ke menu ip disamping lalu pilih DHCP client,klik tanda + lalu pada DHCP dan pada interface pilih yang wlan1 lalu klik ok

3.Masuk ke menu ip lagi lalu klik address lalu pada bagian addres list klik tanda + setelah itu tambahka ip disini saya kasi IP 192.168.100.1/24 lalu interface nya ether2 setela itu klik ok

4.Masuk ke wireless tables lagi lalu klik 2 kali pada wlan 1 dan konfigurasi mode nya ke station lalu band nya 2GHz-B/G/N setelah itu klik scan setelah menekan scan lalu kllik start.pada wlan1.setelah itu pilih wifi mana yang mau dihubungkan,disini saya menggunakan OPPO A12,lalu klik connect

5.Masuk ke wireless tables lagi lalu pergi ke security profile,klik dua kali pada default,pada mode nya pilih yang dynamic keys,lalu centang WPA PSK,WPA PSK dan WPA2 EAP.dibawahnya centang aja semua

6.Masukkan password wifi oppo a12 yang akan dihubungkan,lalu klik ok,jika sudah klik ok,lalu klik tanda tambah lagi untuk membuat password baru,name proile1 lalu ikuti Langkah yang tadi,untuk password wifi nya disini saya kasi aziziaqnasxiitkj2 lalu klik ok.

7.masuk ke wireless tables lagi,masuk ke wifi interface.klik tanda tambah,lalu pilih yang virtual,untuk mode nya ap bridge,SSD nama untuk wifi disini saya kasi AZIZIAQNAS,security profile pilih yang profile1,lalu klik ok

8.pergi ke IP lalu addres lalu klik tanda + ini addres disini saya beri IP nya 192.168.200.1/24.interface wlan2 lalu klik ok

9.Buka IP lagi lalu cari dan pilih firewall,pilih NAT lalu klik tanda +.pilih action lalu pada action pilih yang masquerade,perge ke general pada out.interface pilih wlan2 lalu klik ok

10.pergi Ke IP pilih DNS,pada DNS setting,centangkan pada allow remote requests lalu kilk ok

11.pergi ke IP lagi lalu klik DHCP pada menu DHCP klik DHCP setup,pada bagian DHCP server interface pilih ether2 jika sudah klik next sampai selesai.klik lagi DHCP setup lalu pada DHCP server interface pilih yang wlan 2.lalu klik next sampai selesai.

12.tes apakah jaringan sudah didapatkan yaitu pergi ke new terminal di pojok kanan bawah,lalu ketik googl.com lalu enter.jika tampilan seperti dibawah maka jaringannya sudah dapat atau berhasil

13.Perge ke IP lagi lalu ke firewall pilih NAT klik 2 kali dibawah action.pada bagian out.interface pilih yang wlan1.lalu kik ok,selesai,kalian mengetes jaringan wifinya di hp.
